# TaskApp

## Aplicativo de organização de Tarefas baseado no método Kanban!

![taskappGif](https://user-images.githubusercontent.com/66075182/85160781-2dd27700-b235-11ea-93f9-66a246b9ba47.gif)

## Tecnologias
* ReactJs

## Para rodar na sua máquina
```shell
  # Instalar Dependencias:
  -yarn
  
  # Iniciar o Servidor:
  -yarn start
```
## Problemas Solucionados
* Não poder arrastar os cards para áreas vazias.
